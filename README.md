# PageGuide Showcase

Static project website for PageGuide, a browser extension for grounded, verifiable, and user-controlled web assistance.

## Overview

This site presents PageGuide as a research-style project page built with plain HTML, CSS, and JavaScript for easy GitHub Pages deployment.

PageGuide supports three interaction modes:

- `Find`: highlights exact supporting evidence on webpages, PDFs, and images
- `Guide`: provides step-by-step task help while keeping the user in control
- `Hide`: suppresses distracting or irrelevant content with reversible controls
